== Description ==

Adds a retweet button to all WordPress posts in your blog and update your twitter account with post from your blog.

== Installation ==

How to install Zip.li Retweet

e.g.

1. Upload `zipli-retweet` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'Zip.li Retweet' menu in WordPress

== Frequently Asked Questions ==

= Why to use this plugin? =

Because it integrate your WordPress posts blog with Twitter allowing your readers retweet them.
